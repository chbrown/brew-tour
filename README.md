# brew-tour

## Installation

**From npm:**

```sh
npm install brew-tour
```

**Or github:**

```sh
git clone https://github.com/chbrown/brew-tour.git
cd brew-tour
npm install -g
```

**Import:**

```js
var brew-tour = require('brew-tour');
```
